# Portfolio

This is my personal portfolio website, showcasing my projects, skills, and achievements. The site is built using **HTML, CSS, and JavaScript** and is fully static, making it fast and easy to deploy.

## Live Demo

You can view the live demo of the portfolio here:  
[Portfolio](https://salmon-desert-01cdea703.4.azurestaticapps.net/)


## Features

- Clean and responsive design
- Interactive elements using JavaScript
- Downloadable CV
- Showcase of projects with live links and descriptions
- Contact links and social media integration


## Project Structure

- **index.html**: The main HTML file for the portfolio.
- **script.js**: The JavaScript file containing interactive functionality.
- **style.css**: The main CSS file for styling the portfolio.
- **assets/**: A folder containing images used in the portfolio app.
- **Downloadable CV**: The portfolio includes a link to download CV in PDF format.  


## Getting Started

1. Clone this repository:

    ```bash
        git clone https://github.com/erdisonmalko/Portfolio.git

2. Open the `app/assets/index.html` file in your preferred web browser.  
   - No server or build step is required, as this is a static site.
